# Terraform AWS S3 – Study Project

Projeto de estudo para aprender os fundamentos do Terraform usando AWS.

## O que este projeto cria
- Um bucket S3 na AWS
- Configurado via variáveis
- Com outputs para nome e ARN

## Pré-requisitos
- Terraform CLI
- AWS CLI configurado
- Conta AWS

## Como rodar

terraform init  
terraform plan  
terraform apply  

Para remover os recursos:

terraform destroy  

⚠️ Projeto apenas para fins de estudo.
